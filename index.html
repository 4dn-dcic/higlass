HiGlass: Web based massive matrix display

HiGlass is a tool for displaying and comparing large matrices within a web page. The example below demonstrates its use with contact matrices obtained from chromosome capture assays (HiC). The matrices in the upper row show data from two different cell lines (HMEC and HUVEC). The bottom row shows the ratio of the values from the HMEC data to the values from the HUVEC data.

Source code

The examples above rely on three software packages:
  • Clodius - Tile matrix data
  • HiGlass - Display large tiled 2D matrices
  • Goomba - Zoomable gene annotations and wiggle tracks
